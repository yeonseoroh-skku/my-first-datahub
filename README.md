# 🎓 Hello SKKU Streamlit App
A minimalist, high-performance web application built with Streamlit to greet the Sungkyunkwan University community. This project serves as a foundational template for deploying Python-based data scripts into interactive web interfaces.

## 🌟 Features
Lightweight UI: Built using Streamlit's reactive framework.

SKKU Themed: Custom greeting for the SKKU community.

Instant Deployment: Ready to be hosted on Streamlit Cloud or Heroku.

🛠️ How to Run
Follow these steps to get the application running on your local machine.

1. Prerequisites
Ensure you have Python installed (version 3.7 or higher is recommended).

2. Installation
Clone this repository and install the required dependency:

Bash
## Clone the repository
git clone https://github.com/your-username/hello-skku.git
cd hello-skku

# Install Streamlit
pip install streamlit
3. Launch the App
Run the following command in your terminal:

Bash
streamlit run app.py
The app should automatically open in your browser at http://localhost:8501.
